Please note that the Spring Boot Actuator has already been added to the project

02.
In the application.properties
management.endpoints.web.include=<value>
value could be the comma-separated list of endpoint ids or *
* is considered to be a security risk
Try these urls in the browser:
    http://localhost:8080/actuator/health
    http://localhost:8080/actuator/metrics
    http://localhost:8080/actuator/metrics/jvm.memory.used
    http://localhost:8080/actuator/loggers
    http://localhost:8080/actuator/configprops

05.
To check custom metric,
1. send GET request to: localhost:8080/api/v1/sample
2. check the custom metric: localhost:8080/actuator/metrics/long.running.service.timer
